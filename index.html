Graded Assignment - Personal Web Pages

2023
Adobe Illustrator, Adobe Photoshop


I was tasked to create a personal web page for my graded assignment in Y1S2. I used Adobe Illustrator to create the pages. I decided to go for something minimal yet intriguing. I used shades of blue on top of a black background, which creates a subtle contrast. The main focus of the page is my logo, which I created using Adobe Photoshop. I wanted to go for a 3D metallic-chrome look, so I applied effects such as bevel and emboss, and inner shadow. I also added a curve layer to really achieve the look that I wanted.

Graded Assignment - Mobile Portfolio Prototype

2023
Adobe XD


For my graded assignment in Y2S1, I was tasked to create a mobile portfolio prototype using Adobe XD. My mobile portfolio takes inspiration from the Hong Kong film director, Wong Kar Wai. I wanted to create a portfolio that incorporates his use of bold and saturated colours in his films. Using the split complementary colour scheme, I used different hues of red, green, and blue to create the same captivating and artistic touch to my portfolio.

À Deux - Restaurant Website Prototype

2023
Adobe XD


I was tasked to create a website portfolio for a restaurant using Adobe XD. As I am creating the website for a French restaurant, my overall website design takes inspiration from the neutral colour scheme of Paris. Using an analogous colour scheme, I used shades of beige, brown, and white to reflect the colours of the Parisian streets.

MINDS - Mobile App Prototype

2023
Adobe XD


I was tasked to create a website for MINDS that aid in the communication and understanding of emotions for individuals with intellectual disabilities. I used Adobe XD to create the mobile app prototype. I wanted to create a mobile app that is user-friendly and easy to navigate. I used a monochromatic colour scheme, using different shades of blue to create a calming and soothing effect. I also used icons to represent the different emotions, making it easier for the users to understand and navigate the app.

Graded Assignment - School Library Web App

2023
HTML, CSS, Bootstrap, JavaScript


For my graded assignment in Y2S2, I was tasked to create a school library web app using HTML, CSS, and JavaScript. I wanted to create a visually appealing yet functional web app that is easy to navigate. I used mainly usded green, black, and white to create a clean and professional look. I used fullpage.js, tippy.js, anime.js, and datatables.js to create the different functionalities of the web app.
